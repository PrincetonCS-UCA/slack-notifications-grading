# US CDC SNS Jynneos 2022 Vaccine Distribution data

This is a git scraper, based off of [the `basic-git-scraper` template](https://github.com/jlumbroso/basic-git-scraper-template/).

It collects the data on the federal distribution of the Jynneos vaccine to states during the 2022 Global Monkeypox (MPX) pandemic.
